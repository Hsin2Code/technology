==  No datagram ready ==
A -> A : Read
activate A
A -> K : System call - kernel context switch
activate K
K ->] : Initiate read I/O
deactivate K
note over A : Application blocked
     K <-] : Read response
activate K
== Datagram ready ==
A <- K : Data movement from kernel space to user space
deactivate K
A -> A : Return
deactivate A
